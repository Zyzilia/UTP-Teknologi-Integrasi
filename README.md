Daftar Endpoint
* `GET /api/items` - Menampilkan semua barang.
* `POST /api/items` - Menambah barang baru (otomatis membuat ID unik).
* `GET /api/items/{id}` - Menampilkan detail barang berdasarkan ID.
* `PUT /api/items/{id}` - Mengubah seluruh data barang.
* `PATCH /api/items/{id}` - Mengubah sebagian data (nama atau harga saja).
* `DELETE /api/items/{id}` - Menghapus barang.
